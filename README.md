# eight_puzzle_game

This is a flutter game/project based on the famous 8 puzzle problem where you can swipe the no. '0' in 4 directions(up, down, left, right) to obtain the goal state.

<b>Goal State:-</b>
<br>1   2   3
<br>4   5   6
<br>7   8   0
